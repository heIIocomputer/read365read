# 🌙 밤하늘 독서부 — 반별 독서 현황

학교 도서관 대출 이력(CSV)을 브라우저에서 열어 **반별 독서 현황**을 주간/월간/학기/연간으로 보는 단일 페이지 웹앱입니다.

## 특징
- **집계 방식**: 점수(1인 1점) · 대출 건수 · 반 활동
- **기간**: 주간(기본) · 월간 · 학기 · 연간, 그리고 시작~종료 날짜 직접 설정
- **반 랭킹 정렬**, 히트맵, 반 클릭 시 학생별 상세
- **CSV/엑셀 내보내기**
- **업무 메신저 알림글 작성**: 머리말·꼬리말 + 학년별 순위표(전주 대비 ▲▼) 자동 생성, 코드블록 정렬 옵션

## 사용법
1. 페이지를 엽니다.
2. 상단 **📎 CSV 첨부** 버튼을 누르거나 화면에 CSV 파일을 끌어다 놓습니다.
3. 표가 만들어지면 기간·집계 방식·필터를 조절합니다.

> 개인정보 보호를 위해 대출 이력 CSV는 저장소에 포함되지 않습니다.
> 모든 처리는 방문자 브라우저 안에서만 이루어지며 어떤 데이터도 서버로 전송되지 않습니다.

## 컬럼 형식
`등급, 학년, 반, 대출증번호, 이름, 서명(자료명), 저자, 출판사, 대출일자(YYYY-MM-DD)` 를 포함한 대출 이력 CSV(UTF-8 또는 EUC-KR)를 지원합니다.
